# LinkShortener

**LinkShortener** — это простой и удобный сервис для сокращения длинных URL-ссылок. Проект позволяет преобразовывать громоздкие ссылки в короткие и легко распространяемые, а также управлять ими.

## 🚀 Возможности
- Сокращение длинных URL
- Генерация уникальных коротких ссылок
- Перенаправление на оригинальный адрес


## 🛠️ Технологии
- Backend: Go

## 📌 Применение
- Упрощение обмена ссылками
- Использование в социальных сетях

## ⚙️ Запуск проекта
go run main.go
